LLM 프롬프트 필터

1. 크롬 확장 프로그램 형식으로 제작
    1. 옆에 작은 창을 띄워서 프롬프트 입력하면, 추천 프롬프트와 문제가 될 수 있는 부분 출력 → 직접 GPT에 입력(or 직접 API 연결해서 자동으로 필터링 후 대답 출력)
2. 기능 1(주)  : 민감 정보 필터링(개인정보) 
3. 기능 2 :  자동으로 개선된 프롬포트 작성
    1. 이미지 생성 시 이전 프롬프트에 영향을 받아 이상한 결과 출력 방지
